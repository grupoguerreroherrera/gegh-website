# SIGR · Sistema Integral de Gestión de Restaurante

Sitio web oficial y sistema de gestión operativa para el **Restaurante La Niebla**, Bucaramanga, Colombia.

## Descripción

Aplicación web para gestionar pedidos, reservas, administración de menús, control de caja y generación de reportes. Desarrollada como proyecto académico y real en el marco de la asignatura **Gestión del Software** — Tecnológica del Oriente, Séptimo Semestre.

## Tecnologías

- Frontend: HTML / CSS / JavaScript
- CI: GitHub Actions
- CD: Vercel → gegh-website.vercel.app

## Línea base

- Versión: v1.0.0-baseline
- Rama principal: main
- Despliegue automático desde main vía Vercel
